* WORDPRESS
  * WooCommerce plugin
  * WC Vendors plugin

## STEPS TO CREATE YOUR MARKETPLACE

This is a very abbreviated list to create a multi-vendor marketplace
using wordpress.

* Install WordPress on your server (with SSL)
* Install WooCommerce plugin
* Install WC Vendors plugin
