## METRIC SYSTEM

Measurements,

* kilobyte (KB) equal to 1,024 bytes
* megabyte (MB) equal to 1,024 KB
* gigabyte (GB) equal to 1,024 MB
* terabyte (TB) equal to 1,024 GB
* petabyte (PB) equal to 1,024 TB
* exabyte (EB) equal to 1,024 PB
